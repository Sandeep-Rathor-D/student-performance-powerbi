# student-performance-powerbi
Student Performance Analysis Dashboard using Power BI
This project is developed using Microsoft Power BI to analyze student performance.

🔹 Features
Total Students KPI
Average Percentage
Highest Marks
Pass vs Fail Analysis
Subject-wise Performance
Top 10 Students
Gender-based Analysis
Interactive Filters
🔹 Tools Used
Power BI Desktop
Microsoft Excel
🔹 Output

Interactive dashboard providing insights into student performance.
